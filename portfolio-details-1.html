Introduction of this project

  • Used Spring and Hibernate framework to build a web application for users to shop and order items online.
  • Built a web application based on Spring MVC to support item searching, adding and listing (Dependency injection, Inversion of control , REST API etc)
  • Implemented security workflow via in-memory and JDBC authentication provided by Spring Security.
  • Used Hibernate framework to initialize MYSQL database and provided better support for database operations.
  • Developed a Spring Web Flow to support item ordering and deployed the services to AWS EC2.
